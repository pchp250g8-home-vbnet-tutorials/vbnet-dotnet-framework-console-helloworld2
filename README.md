HelloWorld2. VBNet Tutorials. Platform DotNet FrameWork. Simple Input/Output. Console Application. 
A console application in the VB.Net programming language. 
The program does the following:
  1. Prompts input a name
  2. Displays the phrase "Hello, " plus the name is inputed.
Developed in the Microsoft Visual Studio integrated environment.
